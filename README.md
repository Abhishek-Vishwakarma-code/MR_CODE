# MR_CODE – Master the Art of Problem Solving

MR_CODE is a web-based coding platform designed to help users improve their problem-solving skills through structured practice, real-time feedback, and competitive challenges.

## 🚀 Features

* 🔐 User Authentication (Login / Register / Reset Password)
* 💻 Online Code Editor with multiple language support
* 📊 Real-time code execution & test case evaluation
* 🧠 DSA Problem Bank (Easy / Medium / Hard)
* 🏆 Leaderboard & Contest System
* 📈 Performance Analytics Dashboard

## 🛠️ Tech Stack

* Frontend: Next.js, React, TypeScript
* Backend: Node.js
* Database: MongoDB / MySQL
* Authentication: Firebase
* State Management: Zustand

## 📂 Project Structure

* `/app` → Main application pages
* `/components` → Reusable UI components
* `/lib` → Utility functions & APIs
* `/public` → Static assets

## ⚙️ Getting Started (Local Setup)

Clone the repository:

```bash
git clone https://github.com/your-username/MR_CODE.git
cd MR_CODE
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```
